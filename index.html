DeployR - Simple R Analytics Integration for Application Developers

Data Scientists

Keep producing R scripts, models and data files using your existing analytics tools. Simply upload these files into the DeployR repository to turn them into DeployR -powered analytics Web services.

Application Developers

Use the RBroker framework or one of the client libraries provided in Java, JavaScript or .NET to integrate analytics Web services inside any application, dashboard or backend system.

Enterprise Deployment

Upgrade to take full advantage of flexible enterprise security integration, database high-availability and a scalable grid architecture that supports high-volume and/or high-throughput deployments.

Add the power of R Analytics to any Application, Dashboard, Backend, etc. Get DeployR 8.1.0
