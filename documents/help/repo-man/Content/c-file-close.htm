Closing Files

You can close one file at any time.

To close a file:

  1. Go to the tab of the file you would like to close.
  2. From the Manage menu on the right, choose Close. If there are any unsaved code changes, you will be prompted to save or discard those changes. The tab disappears and the Files tab appears onscreen.

Note: Alternately, hover over the filename in the tab at the top of the screen and click the x.
