Editing File Properties

Only the file properties of the Latest version of a file can be edited since all others in the file history are read-only. Furthermore, only an owner can edit a file.

To edit file properties:

  1. Open the Latest version of a file you own.
  2. Make any desired changes to the following file properties. Changes are automatically saved. Note that you cannot change a file's name or version information.
    • Edit the keywords or file description.
    • Update the list of owners collaborating on this file.
    • Change the access rights on the file to control how the file is deployed.
