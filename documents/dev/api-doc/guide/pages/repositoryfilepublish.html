8.1.5 /r/repository/file/publish

This call publishes a repository-managed file to the public repository.

HTTP Method

POST

Request Format

Content-Type: application/x-www-form-urlencoded; charset=utf-8

Parameters

  • format â€“ specifies markup encoding on call: json or xml
  • repo - specifies the repository name
  • name - specifies the repository file name
  • version - (optional) specifies the repository file version
  • descr - (optional) specifies a description to associate with the published file

Examples 

format=json
repo=testuser
name=hello.txt
version=
descr=Sample publication.

Response Markup (JSON)

{
    "deployr": {
        "response": {
            "success": true,
            "call": "/r/repository/file/publish",
            "repository": {
                "file": {
                    "descr": "Sample publication.",
                    "type": "text/plain",
                    "repo": "public",
                    "size": 12,
                    "name": "hello.txt",
                    "url": "http://host/deployr/r/repository/file/download/
			public/1316681907451/hello.txt",
                    "version": "1316681907451",
                    "ispublic": true
                }
            }
        }
    }
}

Response Markup (XML)

<deployr>
  <response>
    <success>true</success>
    <call>/r/repository/file/publish</call>
    <repository>
      <file>
        <name>hello.txt</name>
        <repo>public</repo>
        <descr>Sample publication.</descr>
        <size>12</size>
        <type>text/plain</type>
        <version>1316681974423</version>
        <ispublic>true</ispublic>
        <url>http://host/deployr/r/repository/file/download/
		public/1316681907451/hello.txt</url>
      </file>
    </repository>
  </response>
</deployr>
