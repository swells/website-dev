18.3 Repository filename, author and version

Previously, R scripts in the repository were identified on the API by a single parameter, named rscript . The rscript parameter is not obsolete. In 6.2, all files in the repository, including R scripts, are now identified by three parameters: filename , author and version . The version parameter is optional:
  • /r/repository/file/list
  • /r/repository/file/upload
  • /r/repository/file/transfer
  • /r/repository/file/write
  • /r/repository/file/update
  • /r/repository/file/revert
  • /r/repository/file/diff
  • /r/repository/file/grant
  • /r/repository/file/download
  • /r/repository/file/delete
  • /r/repository/script/list
  • /r/repository/script/execute

Please refer to the sections Working with Repository Files and Working with Repository Scripts for a detailed description of the repository API.
