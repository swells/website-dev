3.4 Project APIs Summary - Reference Documentation

Authors: Microsoft Corporation

Version: 8.0.0

3.4 Project APIs Summary

API EndpointsDescription
  
Project Management APIs Facilitate project lifecycle management:
  
/r/project/createCreates new temporary, user blackbox or persistent project
/r/project/poolCreates pool of new temporary projects
/r/project/recycleRecycles project R session
/r/project/listLists projects
/r/project/pingPings project
/r/project/aboutRetrieves data about project
/r/project/about/updateUpdates data about project
/r/project/saveSaves project
/r/project/saveasSaves project as a new project
/r/project/closeCloses project
/r/project/grantGrants project to other users
/r/project/importImports project archive
/r/project/exportExports project archive
/r/project/deleteDeletes project
  
Project Execution APIs Facilitate project commands, history and results:
  
/r/project/execute/codeExecutes code on project
/r/project/execute/scriptExecutes script on project
/r/project/execute/interruptInterrupts code execution on project
/r/project/execute/consoleRetrieves latest console output on project
/r/project/execute/historyRetrieves project execution history
/r/project/execute/flushFlushes project execution
/r/project/execute/result/listLists project execution results
/r/project/execute/result/downloadDownloads project execution results
/r/project/execute/result/deleteDeletes project execution results
  
Project Workspace APIs Facilitate working with workspace objects:
  
/r/project/workspace/listLists objects in project workspace
/r/project/workspace/getRetrieves encoded object from project workspace
/r/project/workspace/uploadUploads binary object into project workspace
/r/project/workspace/transferTransfers binary object into project workspace
/r/project/workspace/pushPushes encoded object into project workspace
/r/project/workspace/saveSaves object in project workspace to directory
/r/project/workspace/storeStores object in project workspace to repository
/r/project/workspace/loadLoads repository object into project workspace
/r/project/workspace/deleteDeletes object from project workspace
  
Project Directory APIs Facilitate working with directory files:
  
/r/project/directory/listLists files in project working directory
/r/project/directory/uploadUploads file into project working directory
/r/project/directory/transferTransfers file into project working directory
/r/project/directory/writeWrites text file into project working directory
/r/project/directory/updateUpdates file name/description in project working directory
/r/project/directory/storeStores file in project working directory to repository
/r/project/directory/loadLoads repository file into project working directory
/r/project/directory/downloadDownloads file from project working directory
/r/project/directory/deleteDeletes file from project working directory
  
Project Package APIs Facilitate project package dependency management:
  
/r/project/package/listLists project R package dependencies
/r/project/package/attachAttaches R package dependencies to project
/r/project/package/detachDetaches R package dependencies from project
