6.2.8 /r/project/execute/result/print

This call prints the execution results to a PDF document for the specified project.

By specifying a value for the execution parameter the caller can print only results on the specified executions. By specifying a value for the filename parameter the caller can print a specific result on the specified execution.

HTTP Method

GET

Request Format

Content-Type: application/x-www-form-urlencoded; charset=utf-8

Parameters

  • format – specifies markup encoding on the call: json or xml
  • project – specifies the project identifier
  • execution - (optional) specifies a comma-separated list of execution identifiers
  • filename - (optional) specifies a result file name
  • report - (optional) if true, the PDF document includes summary execution details alongside results
  • landscape - (optional) if true, the PDF document will be printed using landscape orientation

Examples 

format=json
project=PROJECT-c2cbcb44-4c50-4065-9abe-17ae74791838
execution=EXEC-76b503a3-3c2f-4114-b4b2-e3b1b6f88657

Response

Content-Type: application/pdf
