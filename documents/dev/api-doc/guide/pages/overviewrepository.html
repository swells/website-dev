3.6 Repository APIs Summary - Reference Documentation

Authors: Microsoft Corporation

Version: 8.0.0

3.6 Repository APIs Summary

API EndpointsDescription
  
Repository Directory APIs Facilitate working with repository-managed directories:
  
/r/repository/directory/listLists repository-managed files by directory
/r/repository/directory/createCreates a repository-managed directory
/r/repository/directory/renameRenames a repository-managed directory
/r/repository/directory/copyCopies repository-managed files from source to destination directory
/r/repository/directory/moveMoves repository-managed files from source to destination directory
/r/repository/directory/updateUpdates repository-managed file access controls by directory
/r/repository/directory/archiveArchives a repository-managed directory
/r/repository/directory/uploadUploads files to a repository-managed directory
/r/repository/directory/downloadDownloads files from a repository-managed directory
/r/repository/directory/deleteDeletes a repository-managed directory
  
Repository File APIs Facilitate working with repository-managed files:
  
/r/repository/file/listLists repository-managed files
/r/repository/file/fetchFetches the latest meta-data on a file
/r/repository/file/uploadUploads a file to repository
/r/repository/file/transferTransfers a file into repository
/r/repository/file/writeWrites a text file repository
/r/repository/file/updateUpdates properties on a repository file
/r/repository/file/diffDiffs latest file contents against older version
/r/repository/file/revertReverts a repository file to older version
/r/repository/file/grantGrants or revokes ownership of a repository file
/r/repository/file/downloadDownloads a file from repository
/r/repository/file/deleteDeletes files from repository
/r/repository/file/copyCopies files between repository-managed directories
/r/repository/file/moveMoves files between repository-managed directories
  
Repository Script APIs Facilitate working with repository-managed scripts:
  
/r/repository/script/listLists repository-managed scripts
/r/repository/script/executeExecutes a repository-managed script on an anonymous project
/r/repository/script/renderExecutes a repository-managed script on an anonymous project and renders outputs to HTML
/r/repository/script/interruptInterrupts an execution on a HTTP blackbox project
  
Repository Shell APIs Facilitate working with repository-managed shell scripts:
  
/r/repository/shell/executeExecutes a repository-managed shell script on the DeployR server.
