3.5 Job APIs Summary

API EndpointsDescription
  
Job APIs Facilitate working with asynchronous jobs:
  
/r/job/listLists asynchronous jobs
/r/job/submitSubmits asynchronous job
/r/job/scheduleSchedules asynchronous job
/r/job/queryQueries asynchronous jobs
/r/job/cancelCancels asynchronous jobs
/r/job/deleteDeletes asynchronous jobs
