Authors: Microsoft Corporation

Version: 8.0.0

5.2 /r/user/logout

This call signs out the currently authenticated user.

HTTP Method

POST

Request Format

Content-Type: application/x-www-form-urlencoded; charset=utf-8

Parameters

  • format – specifies markup encoding on call: json or xml
  • usercookie – (optional) when specified, value sets application-specific persistent user cookie, which is retrievable on response to /r/user/login call

Examples 

format=json

Response Markup (JSON)

{
    "deployr": {
        "response": {
            "call": "/r/user/logout",
            "success": true,
            "httpcookie": "93FF8F21CDFDC07A01A034C889FC6D83"
        }
    }
}

Response Markup (XML)

<deployr>
  <response>
    <call>/r/user/logout</call>
    <success>true</success>
    <httpcookie>3D69B02F52A0480F2A3BBF3A43ED56C8</httpcookie>
  </response>
</deployr>
